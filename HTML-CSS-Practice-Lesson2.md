# 📋 HTML & CSS Comprehensive Practice Sheet

---

  ## 🏁 1: HTML Structure & Content
  1. [x] **Boilerplate:** Write a full HTML5 boilerplate from memory (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`).
  2. [x] Headings:** Create a page using all heading levels from `<h1>` to `<h6>`.
  3. [ ] Semantic Layout:** Organize a page using `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, and `<footer>`.
  4. **Lists:** 
     [ ] Create an **ordered list** (`<ol>`) of your top 3 goals for this year.
     [ ] Create an **unordered list** (`<ul>`) of 4 programming languages you want to learn.
  5. [ ] Hyperlinks & Images:**
     [ ] Create a link to `https://www.lexicon.se` that opens in a new tab.
     [ ] Insert an image with a descriptive `alt` text.
  6. [ ] Formatting:** Use `<strong>` for bold text and `<em>` for italics.
  7. [ ] Comments:** Add an HTML comment to your code explaining what a specific section does.
  
  ---
  
  ## 🔍 2: Data & Interaction (Tables & Forms)
  1. **Tables:** 
     [ ] Build a 3x3 data table with a header row (`<thead>`) and body (`<tbody>`).
     [ ] **Challenge:** Use `colspan="2"` to merge two header cells.
  2. **Forms:** Create a registration form containing:
     [ ] A `<label>` connected to a text `<input>` for "Username".
     [ ] A password input.
     [ ] A dropdown menu (`<select>`) with at least 3 options.
     [ ] A submit `<button>`.
  3. [ ] **Validation:** Add the `required` attribute to your email input.
  
  ---
  
  ## ✨ 3: Basic Styling (CSS)
  1. ** Selectors:** Create a CSS file and target elements by:
     [ ] **Type** (e.g., `h1`)
     [ ] **Class** (e.g., `.highlight`)
     [ ] **ID** (e.g., `#main-header`)
  2. ** The Box Model:**
     [ ] Create a `div` and give it a specific `width` and `height`.
     [ ] Add 20px of **padding** (internal space).
     [ ] Add 50px of **margin** (external space).
     [ ] Add a 2px solid border with `border-radius: 10px`.
  3. ** Colors & Typography:**
     [ ] Set a background color using a Hex code (e.g., `#f4f4f4`).
     [ ] Change the `font-family` to `sans-serif`.
     [ ] Center your text using `text-align: center`.
  4. [ ] **Box Sizing:** Apply `box-sizing: border-box` to all elements using the universal selector (`*`).
  
  ---
  
  ## 🧱 Optional Task: Layouts (Flexbox & Grid)
  1. [ ] Flexbox Centering:** Use Flexbox to perfectly center a box both horizontally and vertically inside a container.
  2. [ ] Flexbox Navbar:** Create a navigation bar where a "Logo" is on the left and a list of "Links" is on the right using `justify-content: space-between`.
  3. [ ] Flexbox Column:** Create a "Sidebar" layout where items (links) are stacked vertically using `flex-direction: column` with a `15px` gap.
  4. [ ] CSS Grid Basic:** Create a gallery layout with 3 equal columns using `grid-template-columns: repeat(3, 1fr)` and a `20px` gap.
  5. [ ] CSS Grid Spanning:** Create a grid where one item (a "Feature" box) spans across 2 columns using `grid-column: span 2`.
  6. [ ] Hover Effects:** Make a link or button change its color and scale slightly (`transform: scale(1.05)`) smoothly when you hover over it.
  
  ---
  
  ## 🚀 Optional Task: Advanced Challenges
  Ready to go beyond the basics? Try these!
  
  1. [ ] 🧪 Transitions:** Use the `transition` property to make your hover effects animate smoothly over 0.3 seconds.
  2. [ ] 📱 Responsive Design:** Use `auto-fit` and `minmax` in CSS Grid to make a gallery that automatically changes the number of columns based on the screen size.
  3. [ ] 🎨 Custom Variables:** Define CSS Variables (e.g., `--primary-color: #1a73e8;`) in the `:root` and use them throughout your stylesheet.
  4. [ ] 🌓 Dark Mode:** Create a "Dark Mode" version of your practice page using a specific class (e.g., `.dark-theme`) that changes backgrounds to dark gray and text to white.
  5. [ ] ⚓ Smooth Scroll:** Use CSS to make the page scroll smoothly when clicking internal anchor links (e.g., `scroll-behavior: smooth;`).
